# 가차추첨 플랫폼 개발 계획서 v2

## 📋 프로젝트 개요

### 1. **사이트 목적 및 대상 사용자**

- **사이트 목적:**
  - 이 사이트는 온라인 가차 뽑기 참여자들이 '**불투명한 과정에서 오는 불안감 해소하기**'를 목표로, 누구나 직접 검증 가능한 **공정한 추첨(래플) 플랫폼**을 제공합니다.
  - 주최자가 "믿어주세요"가 아닌 "직접 확인해보세요"라고 말할 수 있는 구조
  - 참여자는 조작 의심 없이 "운이 없었다"고 받아들일 수 있는 **공정성 중심의 경험**

- 👥 **대상 사용자**

| 유형                   | 설명                                                            |
| ---------------------- | --------------------------------------------------------------- |
| **가치 지향형 소비자** | 공정한 경쟁과 투명한 프로세스에 매력을 느끼는 사용자            |
| **경험 추구형 사용자** | 단순 구매보다 '추첨 참여' 자체를 즐기는 사용자                  |
| **신뢰 중시형 사용자** | SNS 기반의 불투명한 추첨보다 검증 가능한 시스템을 원하는 사용자 |

### 2. **핵심 기능 및 페이지 구성**

| 구분                  | 기능                                   | 설명                                          |
| --------------------- | -------------------------------------- | --------------------------------------------- |
| **추첨 등록**         | 운영자가 상품, 참여조건, 마감시간 등록 | 상품명, 이미지, 참여기간, seed 생성 기준 등록 |
| **추첨 참여**         | 회원이 로그인 후 참여                  | 1인 1회 제한, 중복방지 로직 적용              |
| **시드 공개**         | 추첨 전·후 외부 데이터 공개            | 예: KOSPI 종가, 날씨 데이터 등 외부 API       |
| **결과 생성 및 검증** | seed + 참여자 명단으로 난수 생성       | 코드 오픈소스로 공개, 누구나 재현 가능        |
| **회원 관리**         | 회원가입/로그인, 마이페이지            | 참여내역, 당첨내역, 배송정보 관리             |
| **배송 관리**         | 운영자용 대시보드                      | 당첨자 배송지 확인 및 배송완료 처리           |

| 페이지                      | 주요 요소                               | 상태    |
| --------------------------- | --------------------------------------- | ------- |
| **홈(Home)**                | 진행 중/마감된 추첨 리스트, 공정성 소개 | ✅ 완료 |
| **추첨 등록 페이지(Admin)** | 상품 등록 폼, 시드 소스 선택            | ✅ 완료 |
| **추첨 상세 페이지**        | 참여 버튼, 참여 조건, 시드 정보         | ✅ 완료 |
| **결과 확인 페이지**        | 당첨자 공개, 검증 코드 보기             | ✅ 완료 |
| **회원 페이지(My)**         | 참여 내역, 배송 정보 입력               | ✅ 완료 |
| **관리자 대시보드**         | 상품/회원/배송 관리 UI                  | ✅ 완료 |

## 🛠 기술 스택

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Data Fetching:** React Query + ky
- **Form:** React Hook Form + Zod
- **State:** Jotai (필요시)
- **Formatting:** ESLint + Prettier
- **Git Hooks:** Husky
- **Package Manager:** pnpm

## 🔧 API 엔드포인트 현황

### 래플 관리 (11/11 완료 ✅)

1. **래플 생성** - `POST /raffles` ✅
2. **래플 목록 조회** - `GET /raffles` ✅
3. **래플 상세 조회** - `GET /raffles/{raffleId}` ✅
4. **래플 잠금** - `POST /raffles/{raffleId}/lock` ✅
5. **래플 취소** - `POST /raffles/{raffleId}/cancel` ✅

### 추첨 및 시드 (3/3 완료 ✅)

1. **추첨 실행** - `POST /raffles/{raffleId}/draw` ✅
2. **추첨 결과 조회** - `GET /raffles/{raffleId}/result` ✅
3. **증빙 정보 조회** - `GET /raffles/{raffleId}/verify/bundle` ✅

### 배송 관리 (3/3 완료 ✅)

1. **배송 정보 제출** - `POST /raffles/{raffleId}/winners/{participantId}/shipping` ✅
2. **배송 정보 수정** - `PATCH /raffles/{raffleId}/winners/{participantId}/shipping` ✅
3. **당첨자 및 배송 현황 조회** - `GET /raffles/{raffleId}/winners` ✅
